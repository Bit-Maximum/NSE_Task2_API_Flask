API Эндпоинты

Эндпоинты городов (/api/v1/cities)

Метод URL Описание Пример
GET /api/v1/cities/ Получить список всех городов Тест
GET /api/v1/cities/<city_id> Получить информацию о городе по ID Тест
POST /api/v1/cities/ Добавить новый город Тело: {"name": "City", "zipcode": 12345}
PUT /api/v1/cities/<city_id> Обновить данные города Тело: {"name": "Updated City", "zipcode": 54321}
DELETE /api/v1/cities/<city_id> Удалить город -
